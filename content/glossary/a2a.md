An open protocol defining how AI agents communicate with each other — delegating tasks, declaring capabilities, and exchanging status updates. Developed by Google; v1.0 released April 2026. IBM's Agent Communication Protocol (ACP) merged into A2A in August 2025. Both are now governed by the Linux Foundation AAIF.

A2A operates at a different layer from MCP. MCP defines how an agent connects to a tool (agent calls tool). A2A defines how agents talk to each other as peers. In a B2A transaction involving multiple agents — a buyer's personal agent, a seller's fulfilment agent, a payment agent — A2A is what allows them to coordinate.

The emerging architecture for agentic systems as of mid-2026: MCP for tool access, A2A for agent coordination, Streamable HTTP as transport.
