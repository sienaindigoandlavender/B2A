The practice of optimising a business's digital presence so that AI agents can discover, understand, and act on it effectively. Coined by Addy Osmani in April 2026 as an extension of SEO thinking into the agent context.

The analogy: SEO optimises for ranking in a search engine's index. AEO optimises for inclusion in an agent's consideration set when it executes a task relevant to your product.

**Osmani's AEO checklist:**

1. Ship an `llms.txt` — curated content map at your domain root
2. Publish a `skill.md` — capability declaration for agent platforms that support tool discovery
3. Audit `robots.txt` — explicitly manage which AI crawlers can access your content
4. Measure tokens — understand how much of your content fits in an agent's context window
5. Monitor agent fingerprints — track agent-specific HTTP user-agent strings in server logs

AEO differs from SEO in one key way: search engines rank and present; agents act. The causality between optimisation and revenue is shorter and more direct.
